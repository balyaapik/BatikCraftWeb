Konfigurasi yang disarankan

  1. Buat bucket privat di Cloudflare R2.
  2. Buat API token Object Read & Write hanya untuk bucket tersebut.
  3. Aktifkan signed URL agar model dan paket sumber tetap privat.
  4. Tekan Uji koneksi & simpan sebelum migrasi file.
Perhatian

Mematikan signed URL membuat file dapat dilayani melalui custom domain publik. Jangan gunakan mode tersebut untuk bucket yang menyimpan .batikmodel atau .batikpack.

Migrasi media lama

python manage.py migrate_media_to_r2 --dry-run
python manage.py migrate_media_to_r2

Perintah pertama hanya memeriksa daftar file. Perintah kedua menyalin file dari folder media VPS ke R2.
